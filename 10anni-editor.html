Come ottengo il token GitHub? (una volta sola)
Il token è una password speciale che permette a questa pagina di salvare le modifiche sul sito al posto tuo. Si crea una volta e resta salvato in questo browser.
  1. Vai su github.com → Fine-grained token (devi essere loggato col tuo account GitHub).
  2. Nome a piacere (es. Editor 10 anni), scadenza lunga.
  3. In Repository access scegli Only select repositories → seleziona santamonicagenova-a11y/SantaWeb.
  4. In Permissions → Repository → Contents imposta Read and write.
  5. Genera, copia il token e incollalo nel campo Token GitHub qui in alto.
Il token resta solo nel tuo browser (non lo vede nessun altro). Se cambi computer lo reincolli.
