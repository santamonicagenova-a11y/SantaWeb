Come si usa

1. Clicca Carica dal sito per caricare automaticamente il menù attivo, oppure Carica da file per selezionarlo dal computer.

2. Modifica prezzi, nomi, asterischi.

3. Clicca Preview per vedere il risultato, poi Pubblica su GitHub per aggiornare il sito.

Foto — Sezione “La nostra cucina”

Seleziona fino a 6 foto. Quelle compilate sostituiranno le foto nella galleria La nostra cucina. La griglia si adatta automaticamente: la prima e l'ultima foto saranno in formato largo.

Menù Vini — Converti da PDF

Seleziona il file PDF della lista vini per convertirlo in HTML e pubblicarlo sul sito. Una preview si aprirà automaticamente prima della pubblicazione.
